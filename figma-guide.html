Introduction to Figma

Figma is a cloud-based design tool primarily used for user interface (UI) and user experience (UX) design. Because it is web-based, it allows for real-time collaboration, much like Google Docs but for design.

The Interface Anatomy

  • Canvas (Center): The infinite workspace where you build your designs.
  • Layers Panel (Left): Organizes all pages, frames, groups, and elements in a hierarchical tree.
  • Properties Panel (Right): Changes based on what you have selected. Allows you to adjust dimensions, colors, typography, constraints, and auto layout.
  • Toolbar (Top): Contains your essential tools (Move, Frame, Shapes, Text, Pen Tool, Comments).

Core Concepts

1. Frames vs. Groups

This is the most important distinction in Figma:

  • Groups (Ctrl/Cmd + G): Simply shrink-wrap around their contents. If you resize a group, the contents stretch and distort.
  • Frames (F): Act like artboards or HTML <div> containers. They have independent sizes, can clip content, and allow you to set constraints and auto-layout. Always use frames for screens (like an iPhone screen) and UI components (like buttons).

2. Auto Layout

Auto Layout is Figma's version of CSS Flexbox. It allows UI elements to grow, shrink, and reflow automatically as their contents change.

  • Fixed Width/Height: The element stays the exact size you set it.
  • Hug Contents: The container shrinks to wrap tightly around the text or objects inside it (perfect for buttons).
  • Fill Container: The element expands to take up all available space within its parent frame.
Pro Tip: Select multiple elements and press Shift + A to instantly wrap them in an Auto Layout frame. This makes spacing perfectly even and easy to adjust.

3. Components & Instances

To maintain consistency (a key UI principle), use Components for UI elements you will reuse, like buttons, navbars, or cards.

  • Main Component (4 solid diamonds): The master version. Changes made here cascade to all copies.
  • Instance (1 hollow diamond): A copy of the main component. You can override text or colors in an instance, but it inherits structure from the Main Component.

4. Styles and Variables

Instead of manually picking the hex code `#0056b3` every time you need a primary blue, you can save it as a Color Style or Variable. You can do the same for Typography (e.g., "Heading 1: Arial 24px Bold") and Effects (e.g., Drop Shadows). Updating the style updates the whole file.

Prototyping

Figma allows you to turn static designs into clickable prototypes without writing code. You access this by switching from the "Design" tab to the "Prototype" tab in the right-hand panel.

  • Connections (Noodles): Blue wires you drag from an element on one frame to another frame.
  • Triggers: How the user interacts (e.g., On Click, On Drag, While Hovering).
  • Actions: What happens next (e.g., Navigate To, Open Overlay, Scroll To).
  • Smart Animate: Figma looks for matching layer names between two frames and automatically creates smooth transitions (fades, slides, scaling) between them.

Essential Keyboard Shortcuts

Mastering these will significantly speed up your design workflow:

  • V : Move tool (Default)
  • F : Frame tool
  • T : Text tool
  • R : Rectangle tool
  • O : Ellipse (Circle) tool
  • Option/Alt + Drag : Duplicate an element
  • Ctrl/Cmd + D : Duplicate in place
  • Ctrl/Cmd + Scroll : Zoom in/out
  • Spacebar + Drag : Pan around the canvas
