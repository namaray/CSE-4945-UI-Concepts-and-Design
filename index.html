Course Materials

1. Perception & Gestalt Principles

  • Mind in Mind (Ch 1-2)
  • Theory

An introduction to the difference between UI and UX. Explore how human perception is shaped by context and goals (Cocktail Party Effect), and dive deep into Gestalt principles like Proximity, Similarity, Continuity, Closure, and Figure-ground.

View Study Guide

2. Visual Hierarchy & Structure

  • Mind in Mind (Ch 3)
  • Design Structure

Learn how our vision is optimized to see structure. This module covers arranging information effectively, removing repetition and noise, using contrast, and establishing clear visual hierarchies in UI controls to make displays easier to scan.

View Study Guide

3. Figma Basics

  • Software Tool
  • Hands-on

A practical introduction to Figma, the industry-standard UI design tool. Learn the fundamentals of setting up frames, navigating the interface, using auto-layout, creating reusable components, and building basic prototypes.
